# シェルコマンド一覧

## pwd

```bash
pwd
```

現在のディレクトリを表示する。

---

## ls

```bash
ls
```

ファイルやフォルダの一覧を表示する。

---

## cd

```bash
cd フォルダ名
```

指定したフォルダへ移動する。

---

## git status

```bash
git status
```

Gitの状態を確認する。

---

## git add .

```bash
git add .
```

変更したファイルをGitに追加する。

---

## git commit

```bash
git commit -m "コメント"
```

変更内容を保存する。

---

## git push

```bash
git push origin main
```

GitHubへアップロードする。
