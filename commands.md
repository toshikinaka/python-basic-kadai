# シェルコマンド一覧

## pwd

```bash
pwd
```

現在のディレクトリを表示する。

---

## ls

```bash
ls
```

ファイルやフォルダの一覧を表示する。

---

## cd

```bash
cd フォルダ名
```

指定したフォルダへ移動する。

---

## git status

```bash
git status
```

Gitの状態を確認する。

---

## git add .

```bash
git add .
```

変更したファイルをGitに追加する。

---

## git commit

```bash
git commit -m "コメント"
```

変更内容を保存する。

---

## git push

```bash
git push origin main
```

GitHubへアップロードする。

## ==0　割り切れる

## and 両方満たす

## elif それ以外で、もし〜なら

## %Y	年（4桁表記）	2015、2022など
## %y	年（2桁表記）	15、22など
## %m	月（先頭に0をつける）	01～12
## %B	月（文字を表示）	April
## %b	月（文字を省略表示）	Apr
## %d	日（月初起点の日数）	01～31
## %j	日（1月1日起点の日数）	222
## %H	24時間単位の時間（先頭に0をつける）	00～23
## %I	12時間単位の時間（先頭に0をつける）	01～12
## %M	分（先頭に0をつける）	00～59
## %S	秒（先頭に0をつける）	00～59
